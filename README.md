### Example Features

-Net 8 Application

-MicroServices Publisher and Consumer APIS.

-RabbitMQ

-Docker

-All configuration mounted in Docker for local.

# How to Run

**Running From**

Using Visual Studio 2022: Chose Docker Compose as start and Run F5 or click on Run.

Using VSCode

`docker Compose up`
